# 🛒 Retail Analytics Data Engineering Project

## 📌 Project Overview

This project demonstrates an end-to-end Retail Analytics Data Engineering pipeline. Raw retail data is extracted, transformed using Python ETL scripts, loaded into SQL Server Data Warehouse, and visualized using Power BI for business insights.

---

- SQL Server
- Python
- Pandas
- pyodbc
- Power BI
- Git & GitHub

## 🔄 ETL Pipeline

1. Extract retail data from source files.
2. Transform data using Python and Pandas.
3. Load cleaned data into SQL Server.
4. Create Dimension and Fact tables.
5. Build an interactive Power BI dashboard.

---

## 📊 Dashboard Features

- Total Revenue
- Total Orders
- Total Customers
- Revenue by Month
- Revenue by Quarter
- Revenue by State
- Top 10 Cities by Revenue
- Product Category Analysis
- Revenue Share by Product Category
- Interactive Filters (Year & State)

---

## 📈 Business Insights

- Identify top-performing product categories.
- Analyze monthly and quarterly sales trends.
- Compare revenue across different states.
- Discover top revenue-generating cities.
- Monitor overall business performance using KPIs.
